# Stock Research Framework

一套系统化的股票投研分析框架，覆盖从基本面到投资决策的 9 个核心维度。

## Overview

这个框架提供了一个全面的股票投资研究模板，适用于任何股票的深度分析。它确保分析全面、逻辑清晰、决策有据。

## 9点分析框架

```
┌─────────────────────────────────────────────────────────────┐
│                    投研分析框架                               │
├─────────────────────────────────────────────────────────────┤
│  1. 公司基本面    2. 财务健康度    3. 竞品对比              │
│  4. 股价走势      5. 宏观环境      6. 市场情绪              │
│  7. 最新财报      8. 成长与风险    9. 投资建议              │
└─────────────────────────────────────────────────────────────┘
```

## 使用方法

这是一个 Claude Code Skill。安装后，在分析任何股票时自动调用这个框架。

### 安装

将 `SKILL.md` 放置在你的 Claude Code skills 目录中。

### 使用

直接询问："帮我分析一下 [股票名称]"，框架会自动引导完成 9 个维度的深度分析。

## 分析维度

### 1. 公司基本面
- 主营业务架构与收入构成
- 行业市场占有率
- 核心竞争壁垒与差异化优势

### 2. 财务健康度
- 营收与净利润趋势
- 毛利率与资产负债结构
- 经营性现金流分析

### 3. 竞品对比
- 近三年营收增速对比
- 市场份额变化
- 盈利水平与估值对比

### 4. 股价走势
- 历史价格节点
- 牛熊周期表现
- 波动率与核心驱动逻辑

### 5. 宏观环境
- 货币政策影响
- 行业监管变化
- 地缘政治与经贸环境

### 6. 市场情绪
- 近期行业新闻
- 券商研报汇总
- 多空情绪倾向

### 7. 最新财报
- 核心财务指标 vs 预期
- 各业务板块表现
- 偏离度分析

### 8. 成长与风险
- 1-3年增长驱动逻辑
- 潜在经营风险
- 中长期估值逻辑

### 9. 投资建议
- 三情景估值区间
- 分类型投资建议
- 核心研判逻辑

## 输出示例

分析完成后会输出包含以下内容的完整报告：

- 📊 数据表格对比
- 🟢🔴 正负面影响标注
- ⚠️ 风险提示
- 💡 具体可操作建议
